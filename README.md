# Sélection de Médiane - Démonstrateur Interactif

## Description
Ce projet implémente et compare différents algorithmes de sélection pour trouver la médiane d'un tableau. Il inclut une interface graphique interactive (Tkinter) permettant de visualiser les étapes de comparaison en temps réel.

## Fonctionnalités
- **Quickselect**: Algorithme de sélection rapide randomisé
- **Médiane Probabiliste**: Algorithme probabiliste moderne
- **Visualisation**: Interface graphique avec navigation pas à pas
- **Démonstrateur**: Comparaison de performance entre algorithmes

## Technologies Utilisées
- **Langage**: Python 3.6+
- **GUI**: Tkinter
- **Analyse**: Matplotlib (pour les graphes de croissance)
- **Architecture**: MVC (Model-View-Controller)

## Utilisation
Lancez le démonstrateur avec :
```bash
python main.py
```
